---

## 项目亮点

- 城市地标识别：根据城市自动提取标志性建筑、文化元素或地理景观
- 天气视觉融合：将实时天气状态融入 3D 微缩场景中
- 艺术化表达：输出具备统一审美风格的视觉海报
- 动态延展：基于静态海报继续生成轻量动态效果
- 适合传播：兼顾工具属性与内容传播属性

## 为什么它不只是天气工具

传统天气产品更强调信息效率，而 SkyVibe AI 尝试把天气、城市地标、地域文化和视觉审美融合成一种可观看、可分享、可延展的内容表达。

它不是简单地展示“今天几度、多云还是晴”，而是在回答另一个问题：

**“如果把一座城市今天的天气，变成一张有氛围感的微缩城市海报，会是什么样？”**

## 使用方式

### 输入示例
- 上海天气
- 伦敦天气
- 东京天气图片
- 北京天气海报
- 伊犁天气

### 输出逻辑
- 明确要求“只要图片 / 海报”时：输出静态海报
- 其他默认情况：输出静态海报 + 动态效果

## 产品思路

我更关心的不是“天气工具还能不能再做一个”，而是：

- AI 是否能把结构化信息转译为高完成度视觉内容
- 天气是否可以成为城市内容生成的入口
- 同一套能力是否可以延展到车机、座舱屏、文旅大屏、城市信息可视化等场景
