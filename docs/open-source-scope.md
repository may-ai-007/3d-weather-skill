# 开源范围说明

## 我公开了什么

本仓库公开的是以下内容：

- 项目背景与设计目标
- 视觉效果示例
- 用户使用说明
- 公开版提示框架
- 场景化产品思考
- 行业应用设想

## 我没有公开什么

为了保护项目的核心积累，以下内容不在本仓库公开：

- 完整系统提示词
- 关键风格控制词
- 私有工作流配置
- 完整 Coze Skill 文件
- 城市要素抽取细节
- 异常处理与失败兜底逻辑
- 动图生成衔接细节

## 为什么这样做

我希望这个仓库能完成三件事：
1. 清楚展示项目价值
2. 分享产品与设计思路
3. 保留核心资产，避免低成本复制

因此，这个仓库采用的是“展示性公开”而不是“完整可复刻公开”。
