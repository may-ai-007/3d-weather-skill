# 参与贡献说明 / Contributing

感谢你关注这个项目。

本仓库是一个以项目展示、产品思考、AI 体验表达和场景探索为主的公开仓库，因此它与传统代码型开源项目的协作方式会有一些不同。

---

## 欢迎哪些贡献

欢迎通过以下方式参与：

- 修正文档中的错别字、病句或表达不清的地方
- 优化 README 的结构和展示效果
- 补充更清晰的公开文档建议
- 提交与项目相关的场景灵感、案例建议或产品思考
- 反馈失效链接、错误图片路径或展示问题
- 提出更适合公开版仓库的组织建议

---

## 暂不开放的内容

以下内容不在本仓库开放贡献范围内：

- 完整私有 Prompt
- 内部工作流逻辑
- `.skill` 导出文件
- `.coze` 配置文件
- 私有调优策略
- 可直接复刻核心能力的内部资产

这些内容已被明确排除在公开仓库之外。

---

## 提交建议

如果你准备提交 PR，请尽量遵守以下原则：

- 一次只解决一个明确问题
- 保持改动范围小而清晰
- 不上传私有文件或敏感内容
- 不引入与项目定位无关的复杂依赖
- 不把本仓库改造成偏离展示定位的通用模板仓库

---

## 更推荐的参与方式

对于这个项目，我更推荐以下参与方式：

- 提交 Issue 反馈问题
- 提交文档优化建议
- 提出新的城市案例或展示思路
- 提出智能座舱、导航、语音、后排屏等方向的场景想法
